# Стилизация кнопок вкладок экрана изучения способностей

## Дата
28 октября 2025

## Проблема
Кнопки "Обычные способности" и "Развитие души" терялись в интерфейсе, были недостаточно выразительными.

## Решение
Добавлена яркая стилизация кнопок вкладок с разными цветами для лучшей визуальной различимости.

## Стилизация кнопок

### Кнопка "Обычные способности"
**Цветовая схема: Синяя**
- **Фон**: `Color(0.2, 0.4, 0.6, 0.8)` - синий
- **Граница**: `Color(0.3, 0.5, 0.7, 1.0)` - светло-синий
- **Текст**: Белый
- **Размер шрифта**: 16px
- **Минимальный размер**: 200x40 пикселей

```gdscript
var normal_style = StyleBoxFlat.new()
normal_style.bg_color = Color(0.2, 0.4, 0.6, 0.8)  # Синий
normal_style.border_color = Color(0.3, 0.5, 0.7, 1.0)
normal_style.border_width_left = 2
normal_style.border_width_right = 2
normal_style.border_width_top = 2
normal_style.border_width_bottom = 2
normal_style.corner_radius_top_left = 8
normal_style.corner_radius_top_right = 8
normal_style.corner_radius_bottom_left = 8
normal_style.corner_radius_bottom_right = 8
```

### Кнопка "Развитие души"
**Цветовая схема: Фиолетовая с золотой границей**
- **Фон**: `Color(0.6, 0.3, 0.7, 0.8)` - фиолетовый
- **Граница**: `Color(0.8, 0.6, 0.2, 1.0)` - золотой
- **Текст**: Белый
- **Размер шрифта**: 16px
- **Минимальный размер**: 200x40 пикселей

```gdscript
var soul_style = StyleBoxFlat.new()
soul_style.bg_color = Color(0.6, 0.3, 0.7, 0.8)  # Фиолетовый
soul_style.border_color = Color(0.8, 0.6, 0.2, 1.0)  # Золотая граница
soul_style.border_width_left = 2
soul_style.border_width_right = 2
soul_style.border_width_top = 2
soul_style.border_width_bottom = 2
soul_style.corner_radius_top_left = 8
soul_style.corner_radius_top_right = 8
soul_style.corner_radius_bottom_left = 8
soul_style.corner_radius_bottom_right = 8
```

## Визуальные особенности

### Общие характеристики:
- **Скругленные углы**: радиус 8 пикселей для всех углов
- **Граница**: 2 пикселя со всех сторон
- **Применяется к состояниям**: normal, hover, pressed (одинаковый стиль для стабильности)

### Цветовая психология:
- **Синий** (Обычные способности): Спокойный, надежный, традиционный
- **Фиолетовый + Золотой** (Развитие души): Магический, премиальный, особенный

## Альтернативные цветовые схемы

Если нужно изменить цвета, вот несколько вариантов:

### Вариант 1: Зеленый + Золотой
```gdscript
// Обычные способности
normal_style.bg_color = Color(0.2, 0.5, 0.3, 0.8)  // Зеленый
normal_style.border_color = Color(0.3, 0.6, 0.4, 1.0)

// Развитие души
soul_style.bg_color = Color(0.7, 0.5, 0.1, 0.8)  // Золотой
soul_style.border_color = Color(0.9, 0.7, 0.3, 1.0)
```

### Вариант 2: Красный + Синий
```gdscript
// Обычные способности
normal_style.bg_color = Color(0.6, 0.2, 0.2, 0.8)  // Красный
normal_style.border_color = Color(0.7, 0.3, 0.3, 1.0)

// Развитие души
soul_style.bg_color = Color(0.2, 0.3, 0.7, 0.8)  // Синий
soul_style.border_color = Color(0.4, 0.5, 0.9, 1.0)
```

### Вариант 3: Оранжевый + Фиолетовый
```gdscript
// Обычные способности
normal_style.bg_color = Color(0.7, 0.4, 0.1, 0.8)  // Оранжевый
normal_style.border_color = Color(0.8, 0.5, 0.2, 1.0)

// Развитие души (текущий)
soul_style.bg_color = Color(0.6, 0.3, 0.7, 0.8)  // Фиолетовый
soul_style.border_color = Color(0.8, 0.6, 0.2, 1.0)  // Золотая граница
```

## Технические детали

### StyleBoxFlat параметры:
- `bg_color` - цвет фона кнопки
- `border_color` - цвет границы
- `border_width_*` - толщина границы с каждой стороны
- `corner_radius_*` - радиус скругления каждого угла

### Theme overrides:
- `normal` - обычное состояние кнопки
- `hover` - при наведении мыши
- `pressed` - при нажатии
- `font_color` - цвет текста
- `font_size` - размер шрифта
