# Дизайн экрана изучения пассивных способностей

## 🎯 Концепция:
Отдельный экран для изучения пассивных способностей с прогрессом по редкости и удобным интерфейсом.

## 📊 Статистика по редкости:
- **Common (Обычные)**: 5/35 изучено
- **Uncommon (Необычные)**: 3/25 изучено  
- **Rare (Редкие)**: 1/20 изучено
- **Epic (Эпические)**: 0/15 изучено
- **Legendary (Легендарные)**: 0/10 изучено
- **Mythic (Мифические)**: 0/5 изучено

## 🎮 Функциональность:

### **1. Прогресс-бары по редкости:**
- Визуальное отображение прогресса изучения
- Цветовая индикация по редкости
- Показ количества изученных/общих способностей

### **2. Фильтрация способностей:**
- По редкости (Все, Common, Uncommon, Rare, Epic, Legendary, Mythic)
- По статусу (Все, Изученные, В процессе, Недоступные)
- По типу (Все, Защитные, Атакующие, Утилитарные, Особые)

### **3. Карточки способностей:**
- Название и редкость с цветовой индикацией
- Описание способности
- Прогресс-бар изучения
- Кнопки "Изучить" и "Подробнее"
- Статус (Изучена, В процессе, Недоступна)

### **4. Детальная информация:**
- Полное описание способности
- Источники изучения (какие враги дают прогресс)
- Требования для изучения
- Эффекты способности

### **5. Дополнительные функции:**
- Сброс прогресса изучения
- Экспорт данных о прогрессе
- Поиск по названию способности

## 🎨 Цветовая схема:

### **Редкость:**
- **Common**: Белый (#FFFFFF)
- **Uncommon**: Зеленый (#00FF00)  
- **Rare**: Синий (#0080FF)
- **Epic**: Фиолетовый (#8000FF)
- **Legendary**: Оранжевый (#FF8000)
- **Mythic**: Красный (#FF0000)

### **Статус:**
- **Изучена**: Зеленый фон
- **В процессе**: Желтый фон
- **Недоступна**: Серый фон

## 📱 Адаптивность:
- Поддержка разных разрешений экрана
- Скроллируемые списки для большого количества способностей
- Компактный режим для мобильных устройств
